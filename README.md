# C++ Simple Calculator

A beginner-friendly calculator application written in C++ that performs basic arithmetic operations through a simple console interface.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles user input through the console

## Technologies Used

* C++
* Standard Library (`iostream`)



## Sample Output
